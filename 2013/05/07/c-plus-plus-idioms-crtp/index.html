C++ Idioms: CRTP

发表于 2013-05-07 分类于 C++

CRTP是奇异递归模板模式(Curiously Recurring Template Pattern)的缩写，是Jim Coplien在1995年提出的C+＋惯用法。尽管已经接近20年过去了，当使用boost, WTL等C+＋库的时候，依然能经常见到它的应用。
